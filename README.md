# Ruboty::Itc

Manage iTunes Connect via Ruboty.

## Install

```ruby
# Gemfile
gem "ruboty-itc"
```

## Usage

```
@ruboty itc current <Apple ID> - Get current application information
```

## Output

```
Name     : <Name>
Bundle ID: <Bundle ID>
SKU      : <SKU>
Apple ID : <Apple ID>
Version  : <Version>
Status   : <Status>
```

## ENV
```
ITC_APPLE_ID - iTunes Connect ID
ITC_PASSWORD - iTunes Connect Password
```
